# 01 - Contexto y objetivos

## Contexto
Se requiere una Pokédex web rápida y usable para consultar Pokémon desde PokeAPI.

## Objetivo MVP
Entregar una app funcional que permita:
1. listar Pokémon en páginas,
2. buscar por nombre,
3. abrir detalle de cada Pokémon.

## Alcance MVP
- Frontend con Next.js + TypeScript.
- Datos desde API pública PokeAPI.
- Docker por defecto para ejecución consistente.
- Pruebas mínimas de smoke/funcionalidad básica.
